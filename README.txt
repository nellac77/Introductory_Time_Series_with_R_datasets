This repo contains the datasets (also R scripts, errata, and solutions) associated with the book:

	Introductory Time Series with R
	by Paul S.P. Cowpertwait and  Andrew V. Metcalfe
	Published by Springer; 2009
	https://www.springer.com/us/book/9780387886978
	
In addition to the original note above, this repo also contains a jupyter notebook containing scripts from the book, along with brief explanations of their use and high-level summary of the topics surrounding these scripts from the book. This is an attempt to practice, brush up, and help others.
